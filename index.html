线性回归模型

样本个数：$m$

样本输入值：$x$

样本输出值：$y$

假设函数：$h_\theta(x)=\theta_0+\theta_1x$

参数：$\theta$ （列向量）

代价函数：$J(\theta)=\frac{1}{2m}\sum\limits{i=1}^{m}(h\theta(x^{(i)}-y^(i))^2$

目标：$min_{\theta} J(\theta)$

梯度下降算法

  • $\theta_j := \theta_j - \alpha\frac{\partial{}}{\partial{\theta_j}}J(\theta_0, \theta_1)$ for j = 0 to 1 同步更新
  • $\thetaj := \theta_j - \alpha \frac{1}{m} \sum \limits{i=1}^{m}(h_\theta(x^{(i)})-y^{(i)})x_j^{(i)}$
  • $\alpha$：学习率

多特征的回归模型

  • $h_\theta(x)=\theta_0+\theta_1x_1+\theta_2x_2+…$

  • $h_\theta(x)=\theta^Tx$

  • 特征缩放和归一
  • 正规方程
    • $\theta = (X^TX)^{-1}X^Ty$
    • 不需要特征缩放

逻辑回归模型

对于分类问题，使用线性回归模型会遇到边界不明确的问题

样本个数：$m$

特征个数：n

样本输入值：$x$

样本输出值：$y \in {0, 1}$

假设函数

  • $h_\theta(x)=g(\theta^Tx) = \frac{1}{1+e^{-\theta^Tx}}$

  • $g(z)=\frac{1}{1+e^{-z}}$

参数：$\theta$ （向量）

代价函数

  • 线性回归的代价函数对于逻辑回归非凸
  • $J(\theta)=\frac{1}{m}\sum\limits{i=1}^{m}Cost(h\theta(x^{(i)}), y^{i})$
  • $Cost(h\theta(x^{(i)}), y^{i})=-log(h\theta(x))$ if y=1

  • $Cost(h\theta(x^{(i)}), y^{i})=-log(1-h\theta(x))$ if y=0

  • $J(\theta) = -[\frac{1}{m}\sum\limits{i=1}^{m}y^{(i)}\log h{\theta}(x^{(i)})+(1-y^{(i)})\log (1-h_\theta(x^{(i)})]$

目标：$min_{\theta} J(\theta)$

梯度下降算法

  • $\theta_j := \theta_j - \alpha\frac{\partial{}}{\partial{\theta_j}}J(\theta)$ for j = 0 to n 同步更新

多分类的逻辑回归

  • $maxih\theta^{(i)}(x)$

非线性假设函数

  • $h_\theta(x)=\theta_0+\theta_1x_1+\theta_2x_2+\theta_3x_1x_2+…$
  • $h_\theta(x)=g(\theta_0+\theta_1x_1+\theta_2x_2+\theta_3x_1x_2+…)$

过拟合

减少特征数量

  • 去掉高阶项

正则化

  • 线性回归

    • 代价函数：$J(\theta) = \frac{1}{2m}[\sum\limits{i=1}^{m}(h{\theta}(x^{(i)})-y^{(i)})^2+\lambda\sum\limits_{i=1}^m\theta_j^2]$
    • 梯度下降：$\thetaj := \theta_j - \alpha [\frac{1}{m} \sum \limits{i=1}^{m}(h_\theta(x^{(i)})-y^{(i)})x_j^{(i)} + \frac{\lambda}{m}\theta_j]$
    • 正规方程：$\theta = (X^TX+\lambda A)^{-1}X^Ty$，A是左上角为0的单位阵

  • 逻辑回归

    • 代价函数：$J(\theta) = -[\frac{1}{m}\sum\limits{i=1}^{m}y^{(i)}\log h{\theta}(x^{(i)})+(1-y^{(i)})\log (1-h\theta(x^{(i)}))] + \frac{\lambda}{2m}\sum\limits{j=1}^{n}\theta_j^2$
    • 梯度下降：$\thetaj := \theta_j - \alpha [\frac{1}{m} \sum \limits{i=1}^{m}(h_\theta(x^{(i)})-y^{(i)})x_j^{(i)} + \frac{\lambda}{m}\theta_j]$

神经网络

简单的三层神经网络

前向传播

$a^{(l)}_{j}$ 第l层的第j个激活值

$a1^{(2)}=g(\Theta{10}^{(1)}x0+\Theta{11}^{(1)}x1+\Theta{12}^{(1)}x2+\Theta{13}^{(1)}x_3)$

$a2^{(2)}=g(\Theta{20}^{(1)}x0+\Theta{21}^{(1)}x1+\Theta{22}^{(1)}x2+\Theta{23}^{(1)}x_3)$

$a3^{(2)}=g(\Theta{30}^{(1)}x0+\Theta{31}^{(1)}x1+\Theta{32}^{(1)}x2+\Theta{33}^{(1)}x_3)$

$h\theta(x)=a_1^{(3)}=g(\Theta{10}^{(2)}a0^{(2)}+\Theta{11}^{(2)}a1^{(2)}+\Theta{12}^{(1)}a2^{(2)}+\Theta{13}^{(1)}a_3^{(2)})$

向量表示

$a^{(1)}=x$

$z^{(2)}=\Theta^{(1)}a^{(1)}$

$a^{(2)}=g(z^{(2)})$ (add $a_{0}^{(2)}$)

$z^{(3)}=\Theta^{(2)}a^{(2)}$

$a^{(3)}=g(z^{(3)})$ (add $a^{(3)}_0$)

$z^{(4)}=\Theta^{(3)}a^{(3)}$

$a^{(4)}=h_{\Theta(x)}g(z^{(4)})$

分类问题

  • 最后一层设多个结点，每个结点对应一个类别，选取值最高的为预测类别

代价函数

  • L层，K个类别，m个样本
  • $J(\theta) = -[\frac{1}{m}\sum\limits{i=1}^{m}\sum\limits{k=1}^{K}y{k}^{(i)}\log h{\theta}(x^{(i)}){k}+(1-y{k}^{(i)})\log (1-h\theta(x^{(i)})_k)] + \frac{\lambda}{2m}\sum\limits{l=1}^{L-1}\sum\limits{i=1}^{s_l}\sum\limits{j=1}^{sl+1}(\theta{ji}^{(l)})^2$

后向传播

$\delta^{(l)}_j$ 代表第l层第j个结点的误差

$\delta^{(4)}_j=a^{(4)}_j-y_j$

$\delta^{(3)}=(\Theta^{(3)})^T\delta^{(4)}.*g^{‘}(z^{(3)})$

$\delta^{(2)}=(\Theta^{(2)})^T\delta^{(3)}.*g^{‘}(z^{(2)})$

$\Delta^{(l)}{ij}:=\Delta^{(l)}{ij}+a^{(l)}_j\delta^{l+1}_i$

$D^{(l)}{ij}:=\frac{1}{m}\Delta；ta^{(l)}{ij}+\lambda\Theta^{(l)}_{ij}$ if $j\ne0$

$D^{(l)}{ij}:=\frac{1}{m}\Delta^{(l)}{ij}$ if $j=0$

D就是J对$\Theta$的偏导

梯度检测

  • $\frac{\partial{}}{\partial{\theta_j}}J(\theta) \approx \frac{J(\theta_1+…+\theta_j+\epsilon+…+\theta_n)-J(\theta_1+…+\theta_j-\epsilon+…+\theta_n)}{2\epsilon}$

  • 梯度作差应为很小的值

随机初始化

  • 初始$\theta$不为0，$\theta\in[-\epsilon,\epsilon]$

假设评估

  • 只使用训练集和测试集，会导致使用测试集选择假设，这是不严谨的
  • 使用训练集、验证集、测试集，使用验证集寻找最佳假设

偏差：欠拟合

方差：过拟合

通过学习曲线判断偏差还是方差

解决方法

  • 获取更多训练数据：解决高方差
  • 尝试更少的特征：解决高方差
  • 增加额外的特征：解决高偏差
  • 增加多项式特征：解决高偏差
  • 降低$\lambda$：解决高偏差
  • 增加$\lambda$：解决高方差

误差分析

交叉验证错误率

不对称性分类的误差评估

  • 训练样本中某个类别的数量比别的类别多很多
  • 使用其他评价标准
    • TP：真实类别为1，预测类别为1
    • TN：真实类别为0，预测类别为0
    • FP：真实类别为0，预测类别为1
    • FN：真实类别为1，预测类别为0
    • 查准率P = TP / （TP+FP）
    • 召回率R = TP / （TP+FN）
    • 平均分数 = (P+R) / 2
    • F值 = $2\frac{PR}{P+R}$

机器学习数据

多特征的模型保证低偏差

大量数据保证低方差

支持向量机

优化目标：$min\theta C\sum\limits{i=1}^{m}[y^{(i)}cost1(\theta^{T}x^{(i)})+(1-y)cost_0(\theta^{T}x^{(i)})]+\frac{1}{2}\sum\limits{i=1}^{n}\theta_j^2$

支持向量机间隔的数学原理

  • 向量内积：$u^Tv=P\cdot||u|| $ ，P是向量v在向量u上的投影
  • 当C不太大时，最小化$||\theta||$
  • $p^{(i)}\cdot||\theta|| \ge 1$，if $y^{(i)}=1$
  • $p^{(i)}\cdot||\theta|| \le -1$，if $y^{(i)}=0$
  • 间隔小了p小了，$||\theta||$就大，因此不会选择

核函数

  • 代替多项式特征

  • 标记点l：训练样本

  • 相似度函数： $fi = similarity(x,l^{(i)}) = exp(-\frac{||x-l^{(i)}||^2}{2\sigma^2}) = exp(-\frac{\sum\limits{j=1}^{n}(x_j-l_j^{(i)})^2}{2\sigma^2})$，x与l接近，f约等于1，x与l不接近，f约等于0

  • 优化目标：$min\theta C\sum\limits{i=1}^{m}[y^{(i)}cost1(\theta^{T}y^{(i)})+(1-y)cost_0(\theta^{T}y^{(i)})]+\frac{1}{2}\sum\limits{i=1}^{n}\theta_j^2$

  • 核函数必须满足默塞尔定理以便能够使用大类优化

SVM参数

  • C大：低偏差，高方差
  • C小：高偏差，低方差
  • $\sigma^2$大：高偏差，低方差
  • $\sigma^2$小：低偏差，高方差

运行SVM

  • 选择参数C

  • 选择核函数

    • 不使用核函数：线性核函数 $y=1$ if $\theta^Tx \ge 0$）
    • 高斯核函数：$f_i=exp(-\frac{||x-l^{(i)}||^2}{2\sigma^2})$，where $l^{(i)}=x^{(i)}$，需要选择$\sigma^2$，通常在特征数较少，样本数较多时使用
    • 多项式核函数：$k(x, l)=(x^Tl+const)^{degree}$，x和l都是非负数时使用

  • 多分类kSVM，y=1,2,3…k

与逻辑回归对比

  • 特征数n比样本数m多：使用逻辑回归，或者线性核函数的SVM
  • 特征数n少，样本数m不是很多：使用高斯核函数的SVM
  • 特征数n少，样本数m多：增加特征后使用逻辑回归或线性核函数的SVM

无监督学习

K均值算法

  • 随机初始化k个簇的聚类中心
  • 聚类分配：对每一个样本计算到聚类中心的距离（距离的平方），选择最小的簇
  • 移动聚类中心：移动聚类中心至每一簇的中心

  • 重复上两步

    K均值算法的优化目标

  • $c^{(i)}$：簇的下标，$\mu_{c^{(i)}}$：簇的聚类中心

  • 代价函数：$J(c^{(1)},…c^{(m)},\mu1,…\mu_K)=\frac{1}{m}\sum\limits{i=1}^{m}||x^{(i)}-\mu_{c^{(i)}}||^2$
  • 优化目标：$\min J(c^{(1)},…c^{(m)},\mu_1,…\mu_K)$

随机初始化

  • 确保 $K<m$，随机选取K个样本，将它们 设为聚类中心
  • 重复多次，选择$J$最小的

选择聚类的数量

  • 可视化样本，观察
  • 画出代价函数-K的折线图，观察转折较大的K

降维

数据压缩：把高维度特征投影到低维度

数据可视化：降维后作图

主成分分析PCA

  • 线性回归是最小化竖直距离，主成分分析是最小化垂直距离
  • 均值标准化：$\muj=\frac{1}{m}\sum\limits{i=1}^{m}x^{(i)}_j$，$x_j^{(i)}=x_j-\mu_j$
  • 特征缩放：$x_j^{(i)} = \frac{x_j^{(i)}-\mu_j}{s_j}$
  • 协方差矩阵：$\Sigma = \frac{1}{m}\sum_{i=1}^{n}(x^{(i)})(x^{(i)})^T$
  • $[U, S, V]=svd(\Sigma)$，提取U的前k个列向量实现降维，得到新的特征z
  • $z = U{reduce}^Tx$，$x{approx}=U{reduce}z$，$U{reduce}$是n*k的矩阵
  • 主成分的数量：$\frac{\frac{1}{m}\sum\limits{i=1}^{m}||x^{(i)}-x{approx}^{(i)}||^2}{\frac{1}{m}\sum\limits_{i=1}^{m}||x^{(i)}||^2} \le 0.01$，选择最小的k使得方差保留99%
  • 上式可由S计算：$1-\frac{\sum\limits{i=1}^{k}S{ii}}{\sum\limits{i=1}^{n}S{ii}} \le 0.01$

应用PCA

  • 数据压缩
    • 减少需要储存的数据容量
    • 加速监督学习：通过$U_{reduce}$把$(x^{(i)},y^{(i)})$变为$(z^{(i)}, y^{(i)})$

  • 数据可视化

  • 不建议使用PCA防止过拟合，因为会丢失信息，建议使用正则化，正则化不会丢失信息

异常检测

给定数据集${x^{(1)},…,x^{(m)}}$，判断$x{test}$是否异常，$x{test}$出现的概率$p(x_{test}) < flag$则异常

高斯分布

  • $x$~$\chi(\mu, \sigma^2)$

  • $p(x;\mu, \sigma^2)=\frac{1}{\sqrt{2\pi}\sigma}exp(-\frac{(x-\mu)^2}{2\mu^2})$

  • 参数估计

    • $\mu=\frac{1}{m}\sum\limits_{i=1}^{m}x^{(i)}$
    • $\sigma^2=\frac{1}{m}\sum\limits_{i=1}^2(x^{(i)}-\mu)^2$

异常检测算法

  • 先画出数据看是否大致符合高斯分布，不符合通过log等函数转换为大致符合高斯分布

  • $p(x)=\prod\limits_{j=1}^{n}p(x_j;\mu_j, \sigma_j^2) < \epsilon $ 时异常

  • 对于正样本少的情况，用正样本拟合$p(x)$，用负样本做交叉验证
  • 对于正样本多的情况，用监督学习

多元高斯分布

  • $p(x;\mu,\Sigma)=\frac{1}{\sqrt{2\pi}\sqrt{|\Sigma|}}exp(-\frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu))$

  • $\mu=\frac{1}{m}\sum\limits{i=1}^{m}x^{(i)} $，$\Sigma = \frac{1}{m}\sum{i=1}^{n}(x^{(i)}-\mu)(x^{(i)}-\mu)^T$，需要保证$m > n$

  • $p(x) < \epsilon$ 时异常
  • 计算代价高

推荐系统

基于内容的推荐系统

  • $n_u$：用户数量，$n_m$：电影数量
  • 对每个用户使用线性回归
  • 给定$x^{(i)}$，学习$\theta^{(1)},…\theta^{(nu)}$：$min{\theta^{(1)},…,\theta^{(nu)}}\frac{1}{2}\sum\limits{j=1}^{nu}\sum\limits{i:r(i,j)=1}((\theta^{(j)})^T-y^{(i,j)})^2+\frac{\lambda}{2}\sum\limits{j=1}^{n_u}\sum\limits{k=1}^{n}(\theta_k^{(j)})^2$

协同过滤

  • 给定$\theta^{(1)},…\theta^{(nu)}$，学习$x^{(i)}$：$min{x^{(1)},…,x^{(nm)}}\frac{1}{2}\sum\limits{i=1}^{nm}\sum\limits{j:r(i,j)=1}((\theta^{(j)})^Tx^{(i)}-y^{(i,j)})^2+\frac{\lambda}{2}\sum\limits{i=1}^{n_m}\sum\limits{k=1}^{n}(x^{(i)}_k)^2$
  • 随机初始化$\theta$，学习$x$，再学习$\theta$，…，直至收敛

可以同时计算$x$和$\theta$

  • $J(x^{(1)},…,x^{(nm)},\theta^{(1)},…,\theta^{(n_u})=\frac{1}{2}\sum\limits{(i,j):r(i,j)=1}((\theta^{(j)})^Tx^{(i)}-y^{(i,j)})^2+\frac{\lambda}{2}\sum\limits{j=1}^{n_u}\sum\limits{k=1}^{n}(\thetak^{(j)})^2+\frac{\lambda}{2}\sum\limits{i=1}^{nm}\sum\limits{k=1}^{n}(x^{(i)}_k)^2$

  • $\min J$

  • 在实际中不设置$x_0$和$\theta_0$

  • 随机初始化$x$和$\theta$

  • 最小化$J$
  • 使用$\theta^Tx$预测

均值归一化

  • 如果一个人没有给任何电影评分，直接预测会全都为0
  • 将其他人的评分减去平均值
  • 预测值为$(\theta^{(j)})^T(x^{(i)})+\mu_i$，这样那个人的预测不全为0

大规模机器学习

batch：一次输入给模型的样本个数

epoch：所有样本通过模型的次数

iteration：一个epoch中包含的batch数量

批量梯度下降 Batch gradient desent

  • 输入所有样本才改变参数（iteration为m）

随机梯度下降 stochastic gradient desent

  • 随机打乱数据集 randomly shuffle dataset
  • 每输入一个样本就改变参数（iteration为1），无须对所有样本求和

判断收敛以及选择学习率

在线学习

  • 得到$(x, y)$
  • 使用$(x, y)$更新$\theta$
